Google Sheets 연결 방법 (최초 1회)

  1. sheets.new 접속 → 새 시트 생성
  2. 상단 메뉴 확장 프로그램 → Apps Script 클릭
  3. 기존 코드 전부 지우고 아래 코드 붙여넣기:
    function doGet(){var v=SpreadsheetApp.getActiveSheet().getRange('A1').getValue();return ContentService.createTextOutput(v||'{}').setMimeType(ContentService.MimeType.JSON);} function doPost(e){SpreadsheetApp.getActiveSheet().getRange('A1').setValue(e.postData.contents);return ContentService.createTextOutput('ok');}
  4. 배포 → 새 배포 → 유형: 웹 앱 → 액세스: 모든 사용자 → 배포
  5. 생성된 웹 앱 URL을 아래에 붙여넣기
배포 후 생성된 URL을 그대로 붙여넣으세요.
